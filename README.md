# 📦 Consulta de CEP - Java + ViaCEP

Projeto desenvolvido em Java com o objetivo de consumir a API ViaCEP e retornar informações de endereço a partir de um CEP informado pelo usuário.

---

## 🚀 Sobre o projeto

Esta aplicação tem como objetivo praticar:

- Consumo de API REST
- Tratamento de dados JSON
- Validação de entrada (CEP)
- Boas práticas em Java

O sistema recebe um CEP, valida o formato e faz uma requisição para a API ViaCEP, retornando os dados do endereço.

---

## ⚙️ Tecnologias utilizadas

- ☕ Java
- 🌐 API ViaCEP
- 📦 JSON (ex: Gson ou nativo)
- 🔗 HttpClient (Java)

---

## ✅ Funcionalidades

- ✅ Validação de CEP (8 dígitos)
- ✅ Remoção de caracteres inválidos
- ✅ Consulta de endereço via API
- ✅ Exibição dos dados no console
- ✅ Tratamento de erro para CEP inválido
